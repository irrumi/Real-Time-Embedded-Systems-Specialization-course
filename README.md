# Real-Time Embedded Systems Specialization — Programming Assignments

Готовый и аккуратно структурированный архив `syslog`-файлов по программным заданиям из специализации **Real-Time Embedded Systems Specialization** (Coursera).

## О репозитории

Этот репозиторий содержит результаты выполнения программных заданий и финального проекта по специализации:

1. **Real-Time Embedded Systems Concepts and Practices** (Course 1 of 4)
2. **Real-Time Embedded Systems Theory and Analysis** (Course 2 of 4)
3. Course 3 (теоретический курс без syslog-материалов в этом репозитории)
4. **Real-Time Project for Embedded Systems** (Course 4 of 4)

> Цель репозитория: хранить **готовые programming assignments** в понятной структуре, чтобы легко находить нужные логи по номеру курса и задания.
